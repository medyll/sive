# S44-03 — TemplatePickerModal Component

## Goal
Modal dialog allowing users to browse and select a template when creating a new document.

## Acceptance Criteria
- [ ] Opens on "New document" action
- [ ] Lists templates grouped by category
- [ ] Preview pane shows content of selected template
- [ ] "Use template" confirms selection; "Start blank" skips
- [ ] Keyboard accessible (arrow keys to navigate, Enter to confirm)
