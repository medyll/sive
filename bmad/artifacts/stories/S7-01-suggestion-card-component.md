# S7-01 — SuggestionCard Component

## Goal
Build a reusable SuggestionCard component displaying category badge, severity indicator, suggestion text, and apply/dismiss actions.

## Acceptance Criteria
- [ ] Renders category badge (grammar, style, clarity, structure)
- [ ] Severity indicator (low/medium/high) with colour coding
- [ ] Apply button calls onApply callback with suggestion text
- [ ] Dismiss button calls onDismiss callback and removes card
