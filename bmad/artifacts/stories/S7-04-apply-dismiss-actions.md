# S7-04 — Apply/Dismiss Actions on SuggestionCard

## Goal
Implement apply and dismiss actions that update both the store and the editor content.

## Acceptance Criteria
- [ ] Apply inserts suggestion text into EditorPanel at cursor position
- [ ] Dismiss removes the card from the visible list
- [ ] Applying a suggestion marks it as applied in the store
- [ ] Undo (Ctrl+Z) reverses an applied suggestion

## Notes
Coordinate with EditorPanel content API introduced in S8.
