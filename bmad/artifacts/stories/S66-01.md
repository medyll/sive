# S66-01 — Social Sharing Buttons

id: S66-01
title: social-sharing-buttons

description: |
  Implement a reusable ShareButton component to allow users to share documents via Copy Link, Twitter, and Facebook. Include accessibility (aria labels, keyboard support), responsive styling, and unit tests.

acceptance_criteria:
  - Share control appears in document header and is keyboard accessible.
  - Clicking opens menu with: Copy link (copies URL + toast), Twitter (opens share URL), Facebook (opens share URL).
  - Unit tests cover rendering, menu toggling, copy-to-clipboard behavior, and a11y attributes.
  - Storybook or component preview added (if project uses it).

estimation: 1 dev day

notes: |
  Start with client-side component; no backend changes expected. Use existing design tokens and toasts.
