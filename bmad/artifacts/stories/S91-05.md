## Notes techniques

### Patterns de migration

**Avant :**
```css
.sidebar {
  background: #f9fafb;
  border-right: 1px solid #e5e7eb;
  padding: 16px;
}
```

**Après :**
```css
.sidebar {
  background: var(--color-surface);
  border-right: 1px solid var(--color-border);
  padding: var(--pad-md);
}
```

**Ou avec utilitaires css-base :**
```html
<aside class="bg-surface border-r p-md">...</aside>
```

### data-* attributes (Chrome 139+)

```html
<div data-elevation="md" data-pad="lg" data-radius="md">
  Card with shadow, padding, rounded corners
</div>
```
