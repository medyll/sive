# S22-03 — Suggested Prompts Chips for Empty Chat State

## Goal
Show 3 suggested prompt chips when chat history is empty; clicking one seeds the input and sends.

## Acceptance Criteria
- [ ] 3 SUGGESTED_PROMPTS shown when chatMessages.length === 0
- [ ] Clicking a chip sets input value and immediately sends
- [ ] Chips rendered as prompt-chip buttons

## Notes
SUGGESTED_PROMPTS const array of 3 strings.
