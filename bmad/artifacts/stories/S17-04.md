# Story S17-04 — User profile: display name & avatar initials

**Epic:** Auth
**Sprint:** Sprint 17
**Points:** 2
**Priority:** Could

## User Story

As a **user**, I want my display name to show in the presence list and profile page, so that collaborators can identify me.

## Context

The `users` table has a `name` column. The profile page currently shows the email. This story makes the name editable and surfaces it in the presence list.

## Acceptance Criteria

```gherkin
Given I have a name set in my profile
When I appear in another user's presence list
Then my name (not user ID) is shown as the avatar label

Given I open my profile page
When I edit my display name and save
Then the updated name persists and appears in the toolbar
```

## Technical Notes

- Update `/src/routes/profile/+page.server.ts`: add `updateName` action
- Update profile page Svelte: display name input + save button
- The presence list already uses `user.name` if available

## Out of Scope

- Avatar image upload
- Username uniqueness validation

## Dependencies

- Existing users table with `name` column
- Profile page (Sprint 11)

## Definition of Done

- [ ] Display name editable on profile page
- [ ] Name shown in presence list
- [ ] Unit test covers updateName action
- [ ] PR reviewed and merged
