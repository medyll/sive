# S28-03 — Idle Timeout & Presence Cleanup

## Goal
Auto-mark users as idle after 30s inactivity and clean up offline users after 2 minutes.

## Acceptance Criteria
- [ ] User marked idle after 30s no cursor movement
- [ ] Idle state broadcast to other viewers (remote cursor fades)
- [ ] Offline users cleaned up after 2 minutes without heartbeat
