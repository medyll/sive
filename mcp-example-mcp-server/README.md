# Example MCP Server

Scaffold of a Node/TypeScript MCP server following the MCP TypeScript SDK patterns.

Quick start:

1. Install deps: `pnpm install` or `npm install`
2. Build: `npm run build`
3. Run (stdio): `npm run dev` (uses tsx for development)

This scaffold includes a sample tool registration in `src/index.ts` and an example tool in `src/tools/example_search_users.ts`.
